# Cursor Review Prompt

Act as a senior product-minded reviewer.

Read `AGENTS.md` and the relevant ticket and requirements documents. Review the proposed or completed change for:

- scope compliance;
- false availability or booking claims;
- accessibility;
- responsive behaviour;
- data validation;
- security and privacy;
- SEO impact;
- maintainability;
- unnecessary dependencies;
- tests and edge states;
- consistency with the provider boundaries.

Classify issues as:

- blocker;
- high;
- medium;
- low;
- follow-up.

Do not request future-phase functionality as a Phase 1 defect.
